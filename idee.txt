Ich möchte eine Multiplayer Game System Webbasiert machen


Features:
- Benutzerverwaltung (Benutzer loggen sich mit Benutzername und Passwort ein)
- Rollen (Admin, GameMaster, Spieler)
Admin kann alles, GameMaster kann Spiele Erstellen, Spieler kann nur Spielen beitreten

- Das Core framework soll Frontend und Backend haben
- Per Plugins können Spiele Installiert werden! (Plugin liefert Frontend und Backend code)
- Core muss umfassende API haben(Soll möglichst viel übernehmen wie zb. Auth oder Websockets bereitstellen etc. , damit es für die Spiele Plugins einheitlicher ist.

- Es soll verschiedene Seiten geben.
- Dashboard
- Bibliothek (Dort werden in Kacheln die verfügbaren Spiele angezeigt und wenn man drauf drückt soll eine Erklärung kommen.
- Multiplayer (Gamemaster und Admin können dort neue Spiele erstellen, Spieler können per Lobby Code einer neuen Lobby beitreten)
- Admin (Dort werden alle einstellungen gemacht)


Bedeute Jedes plugin muss in der Bibliothek eine Kachel bereitstellen können und sich im Multiplayer bereich integrieren können. 
Es soll einen Umfassenden Dev Guide mit der API Schnittstelle geben + Regeln die jeder Entwickler einhalten muss!


Spiele Plugins sollen im Admin bereich als gepackte ZIP installiert werden können. Und auch wieder Deinstalliert.
